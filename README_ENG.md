Barber Reservation System
English

Barber System is a web application for managing barber shop reservations.
It allows customers to easily book appointments for available services and time slots.
Barbers can manage their working hours, reservations, and schedule.
The system helps keep all appointments organized in a clear weekly calendar.
The goal of the application is to simplify appointment management for both customers and barbers.

Česky

Barber System je webová aplikace pro správu rezervací v barbershopu.
Zákazníci si mohou jednoduše rezervovat termín pro dostupné služby.
Barbeři mohou spravovat svou pracovní dobu, rezervace a kalendář.
Systém přehledně zobrazuje všechny rezervace v týdenním kalendáři.
Cílem aplikace je zjednodušit správu rezervací pro zákazníky i barbery.
